Cómo abrir la consola para desarrolladores en Chrome

Hay diferentes formas de abrir la consola, todas ellas igual de válidas:

  • En el Menú de Chrome, selecciona la opción "Más herramientas" y dentro de ésta, las "Herramientas para desarrolladores".
  • Haz clic con el botón secundario en cualquier elemento de la página y selecciona Inspeccionar.


Un primer vistazo a los elementos de la consola

Una vez que hemos abierto la consola, vemos un menu en el que podemos seleccionar diferentes pestañas. Vamos a hacer un repaso de cada una de ellas.

  • Elements: En esta opción se muestra el código HTML de la página y las reglas CSS que se han aplicado. Se pueden modificar los elementos y el estilo libremente, aunque se perderán los cambios una vez que recargues la página.
  • Console: La consola de JavaScript puede utilizarse para registrar mensajes que nos ayuden a depurar el código, y también, podemos escribir código y ejecutarlo.
  • Sources: En esta pestaña tenemos todos los archivos que se han cargado para mostrar la página (HTML, CSS, images, etc). Se pueden también añadir puntos de interrupción de la ejecución para depurar el código.
  • Network: En la pestaña Network se mide el tiempo de carga de la página solicitada, y nos permite optimizar el rendimiento de carga.
  • Performance: Esta opción nos permite grabar cómo se comporta la página mientras la utilizamos para detectar problemas de rendimiento.
  • Memory: Este panel proporciona información del uso de memoria de la aplicación. Nos permite detectar posibles fugas de memoria.
  • Application: Este panel muestra información de bases de datos, datos de la sesión, almacenamiento local y cookies.
  • Security: La pestaña Seguridad muestra información sobre la seguridad de la página.
  • Audits: Con esta opción se hace un análisis y se obtiene un informe de recomendaciones que, idealmente, debería cumplir tu página.
